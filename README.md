# 🎬 MovieTime Android

Android Studio 기반 영화 예매 애플리케이션 프로젝트입니다.  
웹 프로젝트(MovieTime PHP 버전)를 기반으로 Android 환경에 맞게 구현한 앱입니다.

> 모바일프로그래밍 과목 개인 프로젝트

---

## 📱 프로젝트 소개

MovieTime Android는 사용자가 영화 정보를 확인하고  
예매 관련 기능을 수행할 수 있도록 설계된 안드로이드 애플리케이션입니다.

본 프로젝트는 다음을 목표로 제작되었습니다:

- Android UI 구성 이해
- Activity 및 화면 전환 구조 학습
- RecyclerView 활용
- 데이터 관리 및 앱 구조 설계

---

## 🚀 주요 기능

- 🎥 영화 목록 화면
- 🔎 영화 상세 정보 보기
- ❤️ 즐겨찾기 기능
- 🎟 예매 화면 구성
- 📱 Activity 간 Intent 데이터 전달
